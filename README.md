# Lógica Matemática para Ingeniería de Sistemas 🚀

Este repositorio contiene mis apuntes, ejercicios y lógica aplicada desarrollados durante mi formación en Ingeniería de Sistemas. El objetivo es documentar la base teórica que sustenta el desarrollo de algoritmos y estructuras de datos.

## 📚 Temas Estudiados

### 1. Variables y Cuantificadores
Entender cómo las sentencias abiertas se transforman en proposiciones mediante la asignación de valores del dominio o el uso de cuantificadores.

* **Cuantificador Universal ($\forall$):** Validaciones globales en conjuntos de datos (Equivalente a métodos como `.every()` en programación).
* **Cuantificador Existencial ($\exists$):** Búsqueda de elementos específicos (Equivalente a métodos como `.some()` o `.find()`).

### 2. Reglas de Inferencia y Deducción
Aplicación de reglas lógicas para garantizar que el flujo de un programa sea válido:
* **Modus Ponens y Modus Tollens:** Fundamentales para la estructura de condicionales `if-else`.
* **Silogismo Disyuntivo:** Lógica de selección y descarte de opciones.
* **Silogismo Hipotético:** Encadenamiento de procesos y dependencias.

### 3. Estructuras Algebraicas en $\mathbb{R}$
Fundamentos de los números reales aplicados al cálculo computacional:
* **Elementos Neutros:** (0 para la suma, 1 para la multiplicación).
* **Inversos:** Propiedades de cancelación y equilibrio en ecuaciones.

## 🛠️ Aplicación en Sistemas
La lógica formal no es solo teoría; es la herramienta que permite:
1. Diseñar consultas SQL precisas.
2. Optimizar condiciones de parada en bucles.
3. Garantizar la seguridad en sistemas de autenticación.
